## CPU/IO 바운드

### CPU 바운드
- CPU의 연산 속도에 의해 작업 속도가 결정되는 작업
- 예: 복잡한 수학 연산, 데이터 분석, 이미지/비디오 처리, 암호화

### I/O 바운드
- 입출력 장치의 속도에 의해 작업 속도가 결정되는 작업
- 예: 파일 읽기/쓰기, 네트워크 통신, 데이터베이스 쿼리

### 차이점
| 구분 | CPU 바운드 | I/O 바운드 |
|------|-----------|-----------|
| 병목 지점 | CPU | 입출력 장치 |
| CPU 사용률 | 높음 | 낮음 |
| 대기 시간 | 적음 | 많음 |
| 최적화 방법 | 멀티프로세싱, 알고리즘 개선 | 비동기 I/O, 멀티스레딩 |

### 최적화 전략
- **CPU 바운드**: 멀티프로세싱을 통해 여러 CPU 코어 활용
- **I/O 바운드**: 비동기 프로그래밍, 멀티스레딩을 통해 I/O 대기 시간 동안 다른 작업 수행
